# agents

This repository contains personal conventions and instructions
for working with automated agents.

It provides a shared baseline that can be referenced by other repositories
to ensure consistent agent behavior.

---

## Usage in Other Repositories

To use these rules in another repository, add an `AGENTS.md` file
to the root of that repository and reference this repository.

### Example

```md
# AGENTS.md

This repository follows the agent conventions defined here:

https://raw.githubusercontent.com/janmarkuslanger/agents/refs/heads/main/AGENTS.md

No repository-specific overrides apply.
```
